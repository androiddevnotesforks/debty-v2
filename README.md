## Features
- Create/Edit/Delete people and movements
- Data persistance with Room
- Theming (Day/Night, MDC)
- App Shortcuts
- App Widgets
- Swipe to delete
- Undo delete
- Animated vector drawables
- Share content
- Image picker
- In app updates / In app reviews
- Unit/Instrumented Testing
- CI using CircleCI

## Libraries Used
- Android KTX
- Material Design Components
- Kotlin Coroutines
- View Binding
- LiveData
- ViewModel
- Navigation
- Room
- Hilt
- Charts
- Play Core
